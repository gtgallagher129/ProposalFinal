Product Proposal

Product Name: NFL Face Off

Team Members:

  • Grant Gallagher

Application Overview:

NFL Face Off is a web-based app that allows NFL fans and fantasy football enthusiasts to compare the statistics of NFL players. Users can easily access player stats, compare their performance.

Technology Overview:

Languages: HTML, CSS, and JavaScript

Libraries: Utilizing NFL APIs to fetch player statistics

Platforms: Web-based, accessible on desktop and mobile devices

Hosting: Azure using Node.js

Proposal Last Updated: [11/06/23]

Feature List

  • Pull stats from NFL APIs to compare players.
  • Pull images of players.
  • Display statistical differences between players.
  • Create charts and display stats side by side in an aesthetically pleasing way.

Future Feature List (non-prioritized):

  • Optimize the app to look great on mobile devices.
  • Star or bookmark players for fantasy enthusiasts.

Basic Technical Features:

  • Use of HTML, CSS, and JavaScript
  • API integration for real-time player statistics.

Advanced Technical Features:

  • Customizable scoring algorithm to calculate player scores based on statistics.
  • Interactive data visualizations, including charts and graphs for in-depth player comparison.
  • User accounts and profiles for saving favorite players and customizing settings.
